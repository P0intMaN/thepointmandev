## Problem Statement

Given two strings `text1` and `text2`, return the **length of their longest common subsequence**. A subsequence is a sequence derived by deleting some elements without changing the relative order. If there is no common subsequence, return 0.

```
Input:  text1 = "abcde", text2 = "ace"
Output: 3  (subsequence "ace")

Input:  text1 = "abc", text2 = "abc"
Output: 3

Input:  text1 = "abc", text2 = "def"
Output: 0
```

## State Definition

`dp[i][j]` = length of LCS of `text1[0..i-1]` and `text2[0..j-1]`.

**If the characters match** (`text1[i-1] == text2[j-1]`):
`dp[i][j] = dp[i-1][j-1] + 1` — extend LCS by 1.

**If they don't match**:
`dp[i][j] = max(dp[i-1][j], dp[i][j-1])` — best of skipping a character from either string.

**Base cases**: `dp[0][*] = 0` and `dp[*][0] = 0` — LCS with an empty string is 0.

## Bottom-Up DP — O(m·n)

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1]; // 1-indexed, dp[0][*] = dp[*][0] = 0

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }

    return dp[m][n];
}
```

## Tracing the Example

`text1 = "abcde"`, `text2 = "ace"`

|   |   | a | c | e |
|---|---|---|---|---|
|   | 0 | 0 | 0 | 0 |
| a | 0 | **1** | 1 | 1 |
| b | 0 | 1 | 1 | 1 |
| c | 0 | 1 | **2** | 2 |
| d | 0 | 1 | 2 | 2 |
| e | 0 | 1 | 2 | **3** |

`dp[5][3] = 3` ✓

| | Time | Space |
|---|---|---|
| 2D DP | O(m·n) | O(m·n) |
| Space-optimized (1 row) | O(m·n) | O(n) |

Space optimization: only the previous row is needed, so maintain two 1D arrays (or one with careful index handling).

## Key Insight

LCS is the foundation of string comparison DP. The two-case recurrence (match vs no-match) is the pattern to memorize:

```
match: dp[i][j] = dp[i-1][j-1] + 1
no match: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
```

This appears in:
- **#72 Edit Distance** — replace `max` with `min`, add cost for substitution
- **#583 Delete Operations for Two Strings** — `m + n - 2 * LCS(s, t)`
- **#1143 LCS itself** — foundation for diff tools (git diff), DNA sequence alignment, spell checking

To reconstruct the actual subsequence (not just the length): backtrack through the DP table — when `dp[i][j] = dp[i-1][j-1] + 1`, the character is part of the LCS; otherwise follow the larger neighbor.

## Edge Cases

- **One empty string**: LCS = 0. `dp[0][*] = 0` handles this.
- **Both strings identical**: LCS = length of either string.
- **No common characters**: every cell takes the `max(dp[i-1][j], dp[i][j-1])` path, returns 0.
- **Identical characters in different order**: LCS finds the longest shared subsequence regardless of position.
